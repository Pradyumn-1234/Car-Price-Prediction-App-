# Car-Price-Prediction-App-
A Machine Learning based Car Price Prediction System built with Python and Streamlit that estimates the price of a car based on features like company, model, year, fuel type, and kilometers driven.


## 🚗 Car Price Prediction System

The application is built using Python and Streamlit, providing an interactive and user-friendly interface where users can input car details and instantly get the estimated price.

### ⚙️ How It Works

The system uses a Linear Regression model to predict car prices. Categorical features like company, car name, and fuel type are converted into numerical form using OneHotEncoder. A machine learning pipeline is used to combine preprocessing and model training efficiently.

### 📊 Features

- Interactive UI using Streamlit  
- Real-time price prediction  
- Data visualization using heatmap  
- Handles categorical data with encoding  
- Clean and responsive design  

### 🧠 Technologies Used

- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- Streamlit  

### 🎯 Output

The system predicts the estimated price of the car based on user input and displays it instantly in a clean and attractive interface.
